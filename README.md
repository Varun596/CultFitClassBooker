# Cult Fit Class Booker

Book your Cult.Fit classes .

### Prerequisites
- Get `apikey` from any request after you login to [Cure.Fit](https://cult.fit/)
- Get `at` cookie from [Cure.Fit](https://cult.fit/)
- Save them in `application.properties`
- Deploy to Heroku
